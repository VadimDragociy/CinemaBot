по умолчанию текст считается запросом

## Про самого бота

1) из библиотек я использовал aiohttp и aiogramm (альтернативы не пробовал ибо эти работают норм), а также aiosqlite
2) бот работает следующим образом - делается запрос по апи на [неофициальное апи кинопоиска](https://api.poiskkino.dev/documentation#/%D0%A4%D0%B8%D0%BB%D1%8C%D0%BC%D1%8B%2C%20%D1%81%D0%B5%D1%80%D0%B8%D0%B0%D0%BB%D1%8B%2C%20%D0%B8%20%D1%82.%D0%B4./MovieController_searchMovieV1_4)

    Оттуда же я беру рейтинг, постер и описание

    Если фильм находится там по запросу пользователя, то все шикарно, идем в вк и там по запросу user_query + фильм ищем наиболее релевантные ссылки (апи вконтакта клянется, что они наиболее релевантные)

    Если фильма на кинопоиске нет, то увы и ах

3) в итоге юзер получает топ 3 результата из вк (хватает по большей части), рейтинг, постер и описание фильма
